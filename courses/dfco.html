Objectives

  • To learn Boolean algebra and simplification of Boolean functions
  • To learn to design and analyze different combinational circuits
  • To study the basics of synchronous sequential logic and analyze and design sequential circuits
  • To understand the important components of a computer system and the basic organization
  • To learn to write code in hardware description languages for designing larger digital systems
